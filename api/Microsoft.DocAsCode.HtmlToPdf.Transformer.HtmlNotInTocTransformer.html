    Transform(IEnumerable<String>)

    1. Retrive from each html.
    2. Foreach link, try to fix it. 2.1 If the link is full path, just keep it. 2.2 If the link is root path(/a/b), just add host in prefix. 2.3 If the link is relative path(a/b.html) 2.3.1 If the link in TOC, just keep it. 2.3.2 If the link NOT in TOC and in Manifest, try to get the canonical url. 2.3.3 If the link NOT in TOC and NOT in Manifest, log warning to the invalid link. 2.4 Others, keep it as the origin.
